boxScore

boxScore is my capstone Rails application. It allows a User to create a basketball team and create and save games, players, and statistics. Those statistics are transformed to give basic information, like field goal percentage, as well as advanced numbers, like true shooting percentage.

boxScore allowed me to put together all of what I learned during the Bloc Rails Web Development course: MVC, Ruby, and test driven development.

  • GitHub: boxScore
  • Heroku: boxScore

Blocipedia

Blocipedia is a replica of Wikipedia. It also allows users to upgrade their account and create private wiki articles.

This application taught me how to integrate a payment API (Stripe) into a Rails application for the first time.

  • GitHub: Blocipedia
  • Heroku: Blocipedia

Bloccit

Bloccit is a replica of Reddit. This was the first app that the Bloc Rails curriculum has students build. Bloccit was my first entree into Rails, MVC, Ruby programming, and TDD.
